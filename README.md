# Help Desk Ticketing Lab

## Overview

This project is a simulated Help Desk Ticketing Lab created using Freshdesk to practice incident management, troubleshooting workflows, technical documentation, and customer support processes.

The objective of this lab was to gain hands-on experience with common IT support scenarios and document the troubleshooting and resolution process in a structured ticketing environment.

> Note: All tickets in this project are simulated for learning and portfolio purposes. No real customer data or production systems were involved.
