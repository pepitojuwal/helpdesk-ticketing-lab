# Ticket #005 - Computer Running Extremely Slow

## Scenario
User reported that their Windows workstation was running significantly slower than normal. Applications were taking several minutes to open, and the system was becoming unresponsive during normal work.

## Investigation
- Verified issue with the user and observed system slowdown
- Checked system performance and identified high resource usage
- Found multiple unnecessary applications running in the background
- Reviewed startup programs and identified non-essential applications launching at startup

## Root Cause
Excessive startup applications and background processes causing high system resource usage.

## Resolution
- Disabled unnecessary startup programs
- Cleared temporary files
- Restarted workstation
- Monitored system performance after reboot

## Outcome
System performance improved significantly. User confirmed that applications were loading normally and the workstation was stable.
